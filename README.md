Sports Tournament Architecture Project
Overview

This repository contains the design and implementation of a Team-Based Soccer Tournament Management System, developed as part of a Software Architecture course project.

The project focuses on architectural design, quality attributes, and trade-off analysis, rather than feature completeness. It demonstrates the use of both monolithic and distributed architectures, implemented using Laravel, MySQL, Docker, and Kubernetes.

Project Objectives

Design a realistic IT system for managing soccer tournaments

Identify and justify key architecture characteristics

Implement:

One monolithic architecture

One distributed (microservices-based) architecture

Provide reproducible deployment using containers
